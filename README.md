# My Flask App

## Setup

1. Clone the repo:

git clone https://github.com/yourusername/my_flask_app.git
cd my_flask_app

2. Venv

python -m venv venv
source venv/bin/activate  # On Windows use `venv/Scripts/activate`

3. pip install -r requirements.txt

4. cp .env.example .env

5. flask run
